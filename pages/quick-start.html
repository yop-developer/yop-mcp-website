YOP MCP

快速开始

仅需几步，让我们开启AI驱动的支付集成之旅

1

创建Python虚拟环境

首先，让我们为YOP MCP创建一个独立的Python虚拟环境：

# 创建虚拟环境
python -m venv yop_mcp_env

# Linux/macOS激活环境
source yop_mcp_env/bin/activate

# Windows激活环境
# yop_mcp_env\Scripts\activate

使用虚拟环境可以避免包依赖冲突，确保YOP MCP在一个干净的环境中运行。

2

安装YOP MCP命令行工具

确保虚拟环境已激活，然后使用uv（推荐）或pip安装yop-mcp：

# 使用uv安装（推荐，更快）
uv pip install yop-mcp

# 或使用pip安装
# pip install yop-mcp

# 验证安装
yop-mcp --version

安装成功后，您应该能看到类似这样的输出：

yop-mcp 0.1.0
3

配置您的AI编程工具

选择您使用的AI工具，添加YOP MCP服务配置：

{
  "mcpServers": {
    "yop-mcp": {
      "command": "uv",
      "args": [
        "yop-mcp"
      ]
    }
  }
}
  • mcpServers: 定义可用的MCP服务
  • yop-mcp: 您为YOP MCP服务定义的名称
  • command: 启动命令（使用uv）
  • args: 传递给命令的参数

Cursor

  1. 打开Cursor设置
  2. 找到"AI工具配置"
  3. 添加MCP配置

Cline

  1. 找到配置目录
  2. 创建MCP配置文件
  3. 粘贴配置内容

RooCode

  1. 进入项目设置
  2. 找到"AI工具集成"
  3. 添加工具配置
4

验证配置

让我们通过一个简单的测试来验证YOP MCP是否已正确配置：

请使用yop-mcp获取易宝支付平台的概览信息

正在调用YOP MCP服务获取平台概览...

{
  "action": "yeepay_yop_overview",
  "result": {
    "platform": "YOP开放平台",
    "version": "3.0",
    "capabilities": [
      "支付服务",
      "账户服务",
      "营销服务",
      // ...更多服务
    ]
  }
}

如果您看到类似上面的响应，说明YOP MCP已经成功配置！
