准备工作

在开始之前， 请确保您已具备

为了顺利完成 YOP MCP 的配置和使用，您需要准备以下 环境和信息

AI编程环境

您需要具备以下任一环境：

  • 支持工具调用的IDE (如Cursor)
  • AI编程工具 (如Cline, RooCode)
  • 通过API调用的LLM (如Claude)

推荐使用支持MCP协议的AI开发工具

Python环境

确保您的系统已安装：

  • Python 3.8 或更高版本
  • 包管理工具 uv (推荐) 或 pip
Terminal
# 检查Python版本 python --version # 安装uv (推荐) curl -LsSf https://astral.sh/uv/install.sh | sh

商户信息

实际对接API时需要：

  • 易宝支付商户编号 (merchantNo)
  • 应用密钥对 (YOP MCP可协助生成)
  • 测试时可使用沙箱环境数据

沙箱环境提供模拟数据，方便开发测试

只需两步，激活您的 AI 集成超能力！

按照以下步骤快速安装和配置 YOP MCP，开启您的 智能集成之旅。 从配置到完成，仅需 5 分钟。

1

安装 YOP MCP 工具

在您的本地开发环境安装 yop-mcp 命令行工具

预计用时：2 分钟 | 需要 Python 3.8+
Terminal
# 方式一：使用 uv (推荐)
uv pip install yop-mcp
# 方式二：使用 pip
pip install yop-mcp
# 验证安装
yop-mcp --version

💡 安装提示

推荐使用 uv 进行安装，速度更快。 如果遇到权限问题，可以添加 --user 参数。

2

配置 AI 工具

在您的 IDE 或 AI 编程工具中添加 MCP 服务器 配置

预计用时：3 分钟 | 支持 Cursor、Claude 等
mcp_config.json
{
"mcpServers": {
"yop-mcp": { // 服务名称，可自定义
"command": "uv",
"args": [ "yop-mcp" ]
}
}
}
💡 其他配置方式：
// 使用 pip 安装的情况
"command": "yop-mcp"
// 指定完整路径
"command": "/path/to/yop-mcp"

🔧 配置说明

配置文件位置因工具而异：

  • • Cursor: 设置 → MCP 服务器
  • • Claude Desktop: ~/.claude/claude_desktop_config.json
  • • 其他工具: 参考相应文档
